# Curso de Node.js con TypesScript  y Express desde cero 

¡Hola a todos! ¡Bienvenidos al curso de Node.js y Express donde aprenderemos a crear una api_rest tipo Spotify! A lo largo de este curso, veremos cómo utilizar el patrón MVC(Modelo Vista Controlador) y trabajar con diferentes tipos de bases de datos relacionales y no relacionales. A través de estos conocimientos, los estudiantes tendrán una visión general de cómo construir aplicaciones web modernas de principio a fin, incluyendo tanto el front-end como el back-end.
